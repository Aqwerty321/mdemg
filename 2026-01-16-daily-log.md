## Key Features Implemented

### CLI Tools
- **`cmd/decay`** - Exponential edge weight decay with pinned edge protection
- **`cmd/consolidate`** - Cluster detection and abstraction promotion
- **`cmd/prune`** - Graph maintenance (weak edge removal, orphan tombstoning, node merging)

### API Endpoints
| Endpoint | Method | Purpose |
|----------|--------|---------|
| `/v1/memory/retrieve` | POST | Vector search + graph expansion |
| `/v1/memory/ingest` | POST | Single memory ingestion |
| `/v1/memory/ingest/batch` | POST | Bulk ingestion (up to 100) |
| `/v1/memory/reflect` | POST | Deep context exploration |
| `/v1/memory/stats` | GET | Per-space statistics |
| `/v1/memory/nodes/{id}/archive` | POST | Soft-delete |
| `/v1/memory/nodes/{id}/unarchive` | POST | Restore |
| `/v1/memory/nodes/{id}` | DELETE | Hard-delete |
| `/v1/metrics` | GET | Graph-wide health metrics |

### Core Algorithms
- **Hebbian Learning**: `Δw = η * a_i * a_j - μ * w_ij`
- **Spreading Activation**: Multi-hop with decay
- **Scoring Formula**: `S = α*V + β*A + γ*R + δ*C - φ*log(1+deg) - κ*d`

### Infrastructure
- Request validation with `go-playground/validator/v10`
- Structured JSON logging middleware
- LRU embedding cache (reduces API calls)
- Anomaly detection (duplicate/stale detection)
